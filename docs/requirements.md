# 根目录需求文档

## 背景

仓库目录结构已经调整为多项目模式，根目录下的每个一级子目录都是一个独立的 Google Apps Script 项目。根目录需要一个统一入口，帮助快速理解各项目职责并跳转到各自文档。

## 目标

1. 在根目录提供一份总项目 `README.md`。
2. 明确说明每个一级子项目是做什么的。
3. 为每个一级子项目提供到其 `README.md` 的直接链接。
4. 约束后续维护者在目录结构变化时同步更新根目录索引。

## 非目标

1. 不在根目录合并各子项目的详细使用说明。
2. 不把各子项目改造成统一构建或统一测试入口。
3. 不修改各子项目的业务逻辑。
