# Translate Slide 需求文档

## 背景

当前仓库把 `translate-slide` 作为独立 Apps Script 子项目管理，需要补齐最基本的项目文档，说明它的用途、边界和用户可见行为。

## 目标

1. 在 Google Slides 打开时提供 `Translate` 菜单。
2. 支持把当前选中的 slide 在中文和英文之间互译。
3. 保留原页面结构，只替换可翻译的文本内容。
4. 为项目提供独立 README，便于从根目录索引进入后快速理解用途。

## 功能需求

1. 用户必须先选中一个 slide，脚本只处理当前页。
2. 脚本需要遍历形状文本和表格单元格文本。
3. 中文转英文时，只翻译检测到中文内容的段落。
4. 英文转中文时，只翻译未检测到中文的段落。
5. URL、邮箱地址、空字符串不应进入翻译流程。
6. 当没有选中 slide 或没有可翻译文本时，需要给出明确提示。

## 非目标

1. 不支持整份 deck 批量翻译。
2. 不支持图片 OCR、备注翻译或复杂对象翻译。
3. 当前不定义术语表、上下文记忆或版式重排策略。
